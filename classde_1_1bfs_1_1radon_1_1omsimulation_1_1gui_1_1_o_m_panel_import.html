Detailed Description

Creates and shows the data import panel for this software tool. Allows the user to convert prepared CSV files to OMB object files used for simulations later on.

void de.bfs.radon.omsimulation.gui.OMPanelImport.actionPerformed ( ActionEvent  evt)
