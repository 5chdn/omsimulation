Creates a panel displaying the distribution chart of certain selected statistical values.

Parameters:
titleThe headline of the chart. Will be hidden if set to null.
statisticsThe selected statistics of a campaign containing all needed values.
roomTypeThe room type to determine the color of the chart.
previewWill hide annotations, labels and headlines if set true.
mouseEventWill enable mouseClickedEvent if set true. Use with care, and only inside the results panel. Set to false if you are unsure what you are doing.
Returns:
A chart displaying the distribution of certain selected statistical values.
